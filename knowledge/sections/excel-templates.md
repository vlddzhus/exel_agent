# EXCEL DESIGN SYSTEM — Template Reference

Палитры и шаблоны оформления для запросов «сделай красиво», «оформи таблицу».
Источник истины для `applyAutoDesign` (auto-designer.ts) и ручного оформления.

---

## Палитра: Professional (Corporate Blue) — универсальная

Применяется по умолчанию для деловых таблиц, отчётов общего назначения.

| Элемент | Значение |
|---|---|
| Заливка шапки | `#2B579A` (темно-синий) |
| Цвет текста шапки | `#FFFFFF` (белый) |
| Чередующиеся строки | `#F2F7FB` / `#FFFFFF` |
| Границы | `#D6E4F0` (thin Continuous) |
| Внешняя рамка | `#2B579A` |
| Шрифт | `Segoe UI`, 10pt тело / 11pt шапка (bold) |
| Итоговая строка | `#1F3864` заливка, `#FFFFFF` текст, bold |

**Когда использовать:** корпоративные отчёты, презентации, деловая переписка.

## Палитра: Financial Report — строгая

Для финансовых отчётов: P&L, балансы, бюджеты, денежные потоки.

| Элемент | Значение |
|---|---|
| Заливка шапки | `#1A1A1A` (чёрный) |
| Цвет текста шапки | `#FFFFFF` |
| Чередующиеся строки | `#F5F5F5` / `#FFFFFF` |
| Границы | `#808080` (thin) |
| Шрифт | `Calibri`, 10pt тело / 11pt шапка (bold) |
| Валюта | `#,##0.00 ₽;[Red]-#,##0.00 ₽` (минус — красным) |
| Проценты | `0.0%` |
| Итоговая строка | `#000000` заливка, `#FFFFFF` текст, bold |

**Числовые форматы по типам колонок:**
- Деньги → `#,##0.00 ₽;[Red]-#,##0.00 ₽` (отрицательные красным)
- Проценты → `0.0%`
- Даты → `DD.MM.YYYY`
- ID/коды → `@` (текст, чтобы не обрезались нули)

**Когда использовать:** финансовые отчёты, бюджеты, инвестиционные расчёты.

## Автоопределение типа таблицы (intent="auto")

`applyAutoDesign` анализирует колонки и выбирает палитру автоматически:

| Профиль | Условие срабатывания | Палитра |
|---|---|---|
| **financial** | Есть деньги/проценты И ≥40% числовых колонок | Financial Report |
| **professional** | Есть ID-колонки И ≥30% числовых | Professional (Blue) |
| **dashboard** | ≥50% числовых колонок ИЛИ есть boolean | Dashboard (Green) |
| **professional** (fallback) | Всё остальное | Professional (Blue) |

### Распознаваемые типы колонок

| Тип | Эвристика | Формат |
|---|---|---|
| `text` | строковые значения | General |
| `number` | числа, не деньги/проценты | `#,##0.00` |
| `currency` | числа ≥ 1000 или со знаками валют | `#,##0.00 ₽;[Red]-...` |
| `percent` | числа в диапазоне (0, 1) | `0.0%` |
| `date` | Date-объекты или паттерны DD.MM.YYYY | `DD.MM.YYYY` |
| `boolean` | true/false, да/нет, yes/no | General |
| `id` | буквенно-цифровые коды ≥4 символов | `@` (текст) |

## Workflow: «Сделай красиво»

Когда пользователь говорит «сделай красиво», «оформи таблицу», «приведи в порядок»:

1. **getWorkbookOverview** — понять, где данные.
2. **applyAutoDesign** (intent="auto") — единый вызов для всей таблицы.
3. Если есть числовые KPI — **applyConditionalFormat** с iconSet (threeTrafficLights1).
4. Если есть дубликаты — **applyConditionalFormat** type="duplicates".
5. **autoFitColumns** — финальная подгонка ширины.

### Альтернатива (вручную, тонкий контроль):

1. **applyCellFormat** на шапку: bold + fill (headerFill) + fontColor (headerFontColor).
2. **applyCellFormat** на тело: fontName, fontSize.
3. **applyNumberFormat** по каждой числовой колонке отдельно.
4. **applyCellFormat** границы.
5. **setColumnWidths** / **autoFitColumns**.
6. (Опц.) **setRowHeights** для шапки (22pt).

Семантические стили для быстрой подсветки (применяются через `applyNamedStyle`):

| Стиль | Цвет | Семантика |
|---|---|---|

| `Output` | оранжевый (`#FCE4D6`) | Результат вычисления |
| `Calculation` | синий (`#D9E1F2`) | Промежуточный расчёт |
| `Total` | серый (`#A5A5A5`) | Итоговая строка |
| `Note` | жёлтый (`#FFFFCC`) | Комментарий |
| `Warning Text` | красный bold | Предупреждение |

## Conditional Formatting шаблоны

### Светофор (KPI)
```
applyConditionalFormat: type="iconSet", iconSet="threeTrafficLights1"
```

### Стрелки (тренд)
```
applyConditionalFormat: type="iconSet", iconSet="threeArrows"
```

### Звёзды (рейтинг)
```
applyConditionalFormat: type="iconSet", iconSet="fiveRating"
```

### Дубликаты
```
applyConditionalFormat: type="duplicates", criteria="duplicateValues", fillColor="#FFC7CE"
```

### Просроченные даты (формула)
```
applyConditionalFormat: type="customFormula", formula="A1<TODAY()-7", fillColor="#FFC7CE", fontColor="#9C0006"
```

### Цветовой градиент (heatmap)
```
applyConditionalFormat: type="colorScale", minColor="#63BE7B", midColor="#FFEB84", maxColor="#F8696B"
```

## Anti-patterns

❌ **Не угадывай цвета.** Если пользователь сказал «сделай красиво» — используй `applyAutoDesign`, а не придумывай палитру.

❌ **Не смешивай палитры.** Все таблицы одного отчёта — в едином стиле.

❌ **Не используй `applyCellFormat` для всей таблицы**, если можно вызвать один `applyAutoDesign`.

❌ **Не забывай autoFit.** После любого форматирования — авто-ширина колонок.

❌ **Чередование строк только для данных.** Шапка, итоги, пустые строки — без чередования.
